Git Cheat Sheet

Create

From existing data:

  1. cd -/my_project_directory
  2. git init
  3. git add .

From existing repository:

  1. git clone -/existing_repo -/new/repo
  2. git clone git://host.org/myproject.git
  3. git clone ssh://user@host.org/project.git

Show

Files changed in working directory

  1. git status

Changes made to tracked files

  1. git diff

What changed between saved version of a file (i.e. ID1 and ID2)

  1. git diff ID1 ID2

History of changes

  1. git log

History of changes for file with diffs

  1. git log -p FILE DIRECTORY

Who changed WHAT and WHEN with a saved file

  1. git blame FILE

A commit identified by ID

  1. git show ID

A specific file from a specific ID

  1. git show ID FILE

All local branches

  1. git branch
