Git Cheat Sheet

Create

From existing data:

  1. cd -/my_project_directory
  2. git init
  3. git add .

From existing repository:

  1. git clone -/existing_repo -/new/repo
  2. git clone git://host.org/myproject.git
  3. git clone ssh://user@host.org/project.git
